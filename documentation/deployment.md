# Deployment Steps

1. Created IAM roles to securely grant permissions to EC2 and Lambda.
2. Launched an EC2 instance using Amazon Linux.
3. Created an S3 bucket for object storage.
4. Deployed an RDS database using a managed engine.
5. Created a Lambda function for serverless processing.
6. Enabled CloudWatch metrics and alarms.
7. Configured SNS for alert notifications.
8. Enabled CloudTrail for auditing AWS API activity.
